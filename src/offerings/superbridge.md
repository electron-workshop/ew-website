<h1 class="h3 fw-bold mt-3 mb-2">Superbridge</h1>
<p class="lead text-secondary mb-4">One conversation, every platform. Messages from any connected app reach everyone else; people just keep using the app they like.</p>

<p>Communities end up segmented by chat platform — some members are on Discord, others on Slack, WhatsApp, Telegram, or Matrix — and conversations fragment along those lines. Superbridge connects those platforms together so a message sent from any one of them reaches everyone else, no matter which app they're in.</p>

<p>It does take some setup. If you run a community group and would benefit from having your community use any chat platform they like while staying connected instead of segmented by platform, we can set this up for you.</p>
